## Change Color

You can change color in index.php file

```
$plugins = [
    new AdminerTheme("default-green")
    ...
];
```

### Available Colors

| Name      | Value            |
| --------- | ---------------- |
| Green     | default-green    |
| Blue      | default-blue     |
| Orange    | default-orange   |
